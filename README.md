# 260123_CityView

260123_CityView is a curated collection of city blockout meshes organized by district, plus a simple three.js viewer for quickly previewing and toggling mesh visibility.

## Features
- District-based organization for quick scene assembly
- OBJ format assets for broad tool compatibility
- Three.js viewer with collapsible district menus and per-mesh visibility toggles
- Double-sided shading for back/front face visibility

## Getting Started
1) Clone the repository.
2) Run `npm install` in the repo root.
3) Start the viewer with `npm run dev`.
4) Open the local dev URL shown in the terminal.

## Controls
- Orbit: left mouse drag
- Pan: right mouse drag (or two-finger drag)
- Zoom: mouse wheel / trackpad pinch
- Mesh visibility: toggle switches in each district menu
